This is a QnA Maker Bot Sample created using Microsoft Bot Framework.

QnA Maker Dialog is to simplify the bot development using QnA Maker knowledge bases. The bot can consume the QnA Maker dialog as shown in the example to forward the conversation to QnA Maker service and post the answer to the user in a seamless way. The package contains QnAMakerService that allows interation with QnA Maker knowledge bases. This service along with the QnA Maker dialog or scorable are useful for building QnA bots in few minutes.

Prerequisites

The minimum prerequisites to run this sample are:

  • The latest update of Visual Studio 2015, 2017. You can download the community version here for free.
  • The Bot Framework Emulator. To install the Bot Framework Emulator, download it from here. Once the emulator has been installed you can debug your sample by folowing the steps provided in this document. Please refer to this documentation article to know more about the Bot Framework Emulator.

SimpleQnABot demonstrates a simple QnA bot where the bot responds with the top answer above specified threshold. One just needs to add the knowledge base id and the subscription key in the Web.config file as shown below.

<appSettings>
...
<add key="QnASubscriptionKey" value="" />
<add key="QnAKbId" value="" />
</appSettings>

And then forward the call from the MessagesController:

public async Task<HttpResponseMessage> Post([FromBody]Activity activity)
{
if (activity.Type == ActivityTypes.Message)
{
await Conversation.SendAsync(activity, () => new Dialogs.SimpleQnADialog());
}
...
...
}
